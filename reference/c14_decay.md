# Radiocarbon decay constants

Decay constants used in the calculation of radiocarbon ages (see
[`c14_age()`](https://c14.joeroe.io/reference/c14_units.md)).

## Usage

``` r
c14_decay_libby

c14_decay_cambridge
```

## Format

A numeric of length 1:

- c14_decay_libby:

  The Libby decay constant, \\\lambda_L=8033^{-1}\\. The standard for
  calculating the conventional radiocarbon age (CRA).

- c14_decay_cambridge:

  The Cambridge decay constant, \\\lambda_C=8267^{-1}\\.

An object of class `numeric` of length 1.

An object of class `numeric` of length 1.
